# ⏱️ Digital Stopwatch using LabVIEW + Python

## 📌 Description
This project demonstrates a digital stopwatch built in LabVIEW and controlled using Python via VI Server.

## 🚀 Features
- Stopwatch UI in LabVIEW
- Python automation to open VI
- Integration using COM (pywin32)

## 🛠️ Tools Used
- LabVIEW
- Python
- pywin32

## ▶️ How to Run
1. Run `Test.py`
2. VI will open automatically
3. Run StopWatch_Panel.vi
